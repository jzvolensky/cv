Short Introduction

Welcome! I'm Juraj.

As a passionate geospatial researcher at Eurac Research in the beautiful Bolzano, Italy, I immerse myself in the dynamic world of Earth Observation, where I dive into complex workflows, scalability, and efficiency. My work revolves around using satellite data and other geospatial sources to understand the alpine environment and make impactful discoveries (while being computationally efficient).

Outside of work you’ll find me exploring new horizons—whether it’s traveling to new destinations, unwinding with friends, or getting lost in the latest video games. I’m also an avid tech enthusiast, always happy to experiment with new programming languages and technologies.

Projects I work on

Here are some of the projects I have been involved in for the last year.

Earth Observation Exploitation Platform Common Architecture (EOEPCA)

EOEPCA is a platform that provides a common architecture for Earth Observation data exploitation. It is a scalable, cloud-based platform that enables users to access, process, and analyze large volumes of Earth Observation data. A key feature is interoperability which is achieved through the use of OGC Application Packages (APs) and the use of the OGC API - Processes standard.

OpenEO

openEO develops an API that allows users to connect to Earth observation cloud back-ends in a simple and unified way. The project maintains the API and process specifications, and an open-source ecosystem with clients and server implementations.

InterTwin

InterTwin is a collaborative project which aims to develop an interdisciplinary Digital Twin Earth (DTE) framework. We have partners from the fields of Earth Observation, physics, astronomy, and computer science.
